About

What is Comtro ?

Comtro is a modern announcement and bell system that simplifies schedules, ensures timely alerts, and offers cloud-based control from anywhere.

COMTRO

Comtro is best & high quality

Comtro uses the PA system which will be commonly available in institutions and other areas for announcements. User can able to record announcements with Comtro Cloud App from anywhere, anytime. User can append this saved file and play automatically for the specified date and time. By using cloud app, the user can add or modify the alarms.

Modern, Simple, and built for smarter communication

There is no need to assign an individual for ringing the bell, inform separate notice or memo to each class or area. User can program various timings for the periods, breaks, etc. Separate modes for schedule multiple bells. You can select the units to rang the alarms and assign separate tones for each bells.
